
---
1. Farm APIs
Method | Endpoint | Parameters | Request Body | Response
POST | /api/farms | None | multipart/form-data: 
- FarmName: string
- FarmDesc: string
- Phone: string
- Area: string
- FarmerId: string($uuid)
- Province, District, Ward, Detail: string
- FarmBanner: string($binary) | 200 OK
GET | /api/farms/me | None | None | 200 OK / 400 if user not authenticated
POST | /api/farms/{farmId}/certificate | farmId (path) | multipart/form-data:
- FarmId: string($uuid)
- Certificate: string($binary) | 200 OK
DELETE | /api/farms/{farmId}/certificate | farmId (path) | None | 200 OK
PUT | /api/farms/{farmId}/certificates | farmId (path) | multipart/form-data:
- FarmId: string($uuid)
- Certificate: string($binary) | 200 OK

2. Season APIs
Method | Endpoint | Parameters | Request Body | Response
GET | /api/seasons | searchTerm (query, optional) | None | 200 OK
POST | /api/seasons | None | application/json:
- seasonName: string
- seasonDesc: string
- startDate, endDate: datetime
- farmId, productId: string($uuid) | 200 OK
GET | /api/seasons/{seasonId} | seasonId (path) | None | 200 OK
PUT | /api/seasons/{seasonId} | seasonId (path) | application/json:
- seasonName, seasonDesc, status: string
- startDate, endDate: datetime | 200 OK
DELETE | /api/seasons/{seasonId} | seasonId (path) | None | 200 OK
PATCH | /api/seasons/{seasonId} | seasonId (path) | None | 200 OK

3. Product Batch APIs
Method | Endpoint | Parameters | Request Body | Response
GET | /api/product-batches/${seasonId} | seasonId (path) | None | 200 OK
POST | /api/product-batches | None | application/json:
- seasonId: string($uuid)
- totalYield, availableQuantity: number
- units: string
- plantingDate: datetime
- isActive: boolean
- price: number | 200 OK
PATCH | /api/product-batches/${batchId} | batchId (path) | application/json:
- soldQuantity: number | 200 OK

4. Category APIs
GET /api/categories
- Get all categories
POST /api/categories
- Create category (multipart/form-data):
  - CategoryName: string
  - CategoryDesc: string
  - IllustractiveImage: file
DELETE /api/categories/{categoryId}
- Delete category by ID

5. Product APIs
GET /api/products?searchTerm&categoryId&location
- Get product list with filters
POST /api/products
- Create product (application/json):
  - productName: string
  - productAttribute: string
  - productDesc: string
  - categoryId: uuid
GET /api/products/{productId}
- Get product details
PUT /api/products/{productId}
- Update product
DELETE /api/products/{productId}
- Delete product
